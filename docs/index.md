# Wet- en Regelgeving LeidenRP

Welkom op de pagina voor de Wet- en Regelgeving van Leiden Roleplay!
In het menu aan de linkerkant vind je alle verschillende documenten die betrekking hebben op Leiden Roleplay.

Zorg ervoor, dat je voor je deelneemt aan Leiden Roleplay, kennis hebt genomen van deze wetten.

- De Algemene Plaatselijke Verordening bevat alle regels die niet te maken hebben met de Roleplay (dit zijn zogezegd de "server regels").
- Het Wetboek Leiden Roleplay bevat alle wetten voor de burgers.
- Alle overige documenten zijn specifiek bedoeld voor bepaalde zaken (zoals een belastingdocument waar je kan opzoeken hoeveel de autobelasting is).
